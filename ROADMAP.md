# Sendity Roadmap

## Vision

Sendity is a privacy-first, self-hosted email platform built on a lightweight PHP framework developed specifically for secure communication applications.

---

# Framework

## ✅ Completed

- PSR-4 Autoloading
- HTTP Request
- HTTP Response
- Router
- Route Parameters
- Dependency Injection Container
- Controller Resolution
- Logging Middleware
- Middleware Pipeline
- Configuration System

## 🚧 In Progress

- Exception Handler

## 📋 Planned

- Event Dispatcher
- Database Layer
- Cache
- Queue
- Scheduler
- Authentication
- CLI Commands
